# 2024 日本楓紅多日行程：動態展示網頁專案

這是一個將 Markdown 格式行程轉換為互動式網頁的輕量級專案，特別針對 2024 年 12 月的日本賞楓旅行設計。

---

## 1. 專案用途
本專案的主要目的是將純文字的旅遊企劃（`tripAll_2024.md`）轉換為美觀且易於操作的網頁界面。它特別優化了行動裝置的閱讀體驗，讓旅行者在旅途中能快速查閱每日景點、飯店住宿資訊，並提供全文檢索功能。

## 2. 使用方法
*   **行程瀏覽**：點擊上方的「Day X」按鈕可切換不同天數的詳細內容。系統會利用 `localStorage` 記錄你最後閱讀的天數，下次開啟網頁時會自動回到該位置。
*   **住宿總覽**：透過「住宿」按鈕可一覽全行程的飯店清單（包含天數、日期與飯店名稱），點擊表格中的任一列可直接跳轉回該天的詳細行程。
*   **關鍵字搜尋**：在「其他資訊」選單中，使用者可輸入關鍵字（例如：抹茶、咖啡、寺廟）搜尋全行程內容，系統會列出所有相符的天數與上下文預覽。

## 3. 核心程式技巧與技術架構
本專案採用純前端技術開發（Vanilla JS + CSS），主要運用以下技術：

### A. 非同步資料處理 (Async/Fetch API)
*   使用 `fetch` 讀取外部 Markdown 檔案。這讓行程內容與展示代碼分離，維護人員只需修改文字檔（`.md`），網頁即會同步更新。

### B. 正規表達式應用 (Regular Expressions)
*   **內容切割**：透過 Regex（例如 `/(?=^## .*D\d+)/m`）自動辨識並分割每一天的行程。
*   **精準擷取**：使用正則表達式從複雜的文字中抓取「住宿：」資訊，並自動清理 Markdown 格式符號。

### C. 內容渲染 (Marked.js Library)
*   整合第三方套件 `marked.min.js`，將純文字的 Markdown 即時轉換為結構化且具備標題、清單、連結的 HTML 內容。

### D. 響應式與現代 CSS 佈局
*   **CSS Variables**：定義主題顏色（如金、紅、米色），實現統一且具質感的日系視覺風格。
*   **Sticky Header**：讓天數選單固定在視窗頂端，方便在長行程中快速導航。
*   **Flexbox**：建立可自動換行且支援水平滾動的按鈕選單，適應各種螢幕尺寸。

### E. 使用者體驗優化 (UX Tools)
*   **Web Storage**：利用 `localStorage` 保持瀏覽進度。
*   **互動式 UI**：搜尋結果與住宿清單皆具備「點擊跳轉」功能，讓資訊查找更加流暢。

---

## 4. 檔案結構
*   `index.html`：網頁骨架與 CSS 樣式定義。
*   `script.js`：核心邏輯（檔案讀取、解析、搜尋與導航）。
*   `tripAll_2024.md`：原始行程資料來源（Markdown 格式）。
*   `background.jpg`：日系氛圍背景圖。
